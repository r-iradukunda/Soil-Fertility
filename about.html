Soil Fertility Analysis System

Key Features

12-Parameter Analysis

Measures N, P, K, pH, EC, OC, and micronutrients (Zn, Fe, Cu, Mn, B, S).

Machine Learning Models

Uses Random Forest and Logistic Regression for accurate classification.

3-Tier Classification

Classifies soil as: 0 (Less Fertile), 1 (Fertile), or 2 (Highly Fertile).

Data Visualization

Generates correlation heatmaps for clear parameter relationships.

How It Works

  1. Data Collection: Gather 12 soil parameters from samples
  2. Data Cleaning: Handle missing values with Random Forest imputation
  3. Model Training: Train classifiers on historical soil data
  4. Classification: Predict fertility level from new soil data

Applications

  • Precision fertilizer application
  • Soil health monitoring over time
  • Agricultural research and education
  • Sustainable farming practices

Our Mission

Provide accurate, accessible soil analysis to help farmers:

  • Optimize crop yields
  • Reduce unnecessary chemical use
  • Practice sustainable agriculture
